# Inspection Layout Management – IATF 16949

Sistema desenvolvido para gerenciar inspeções de layout de produtos, auxiliando no controle e rastreabilidade exigidos pela norma **IATF 16949**.

## Tecnologias

* Java
* Spring Boot
* Spring Data JPA
* Spring Security
* PostgreSQL
* Maven

## Funcionalidades (Em desenvolvimento)

* Cadastro de usuários
* Cadastro de clientes
* Cadastro de produtos
* Cadastro de inspeções de layout
* Consulta de registros
* Atualização de informações
* Exclusão de registros
* Cronograma de inspeção
* Output de relatórios em pdf e csv;
* API REST

## Objetivo

O sistema tem como objetivo centralizar o gerenciamento das inspeções de layout, reduzindo o tempo gasto com processos manuais, aumentando a organização das informações e facilitando a rastreabilidade das inspeções realizadas.
